
Задание по языку XML
1.	В каждый тег <book>  XML- документа <library> (см. файл library.xml) добавить два дополнительных атрибута. Один из них показывает количество экземпляров книги в библиотеке, второй – количество экземпляров в наличии (не выданных читателям).
2.	Составить XSD схему или DTD для проверки правильности документа.
3.	Добавить в указанный документ сведения о новых книгах, всего должно быть не менее 15 книг. Желательно, чтобы у некоторых авторов было по 2-3 книги.
4.	Написать программу на языке Java, реализующую некоторые операции над указанным  XML- документом. Основные требования:
4.1	Графический интерфейс. 
4.2	Проверка на соответствие схеме при загрузке.
4.3	Пользовательское меню.
3.2.1. Вывод всех сведений о книгах в виде таблицы
3.2.2. Добавление новой книги
3.2.3. Поиск книги по автору; году издания; категории
3.2.4 Переоценка отдельных книг (изменение цены)
3.2.4. Эмулирование выдачи книги читателю
4.4. Реализовать указанные пункты меню.
